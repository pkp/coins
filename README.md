# COinS Plugin

- OJS COinS Plugin
- Version: (see version.xml)

About
-----
This plugin embeds OpenURL COinS (http://ocoins.info/) in OJS articles,
permitting tools like Zotero (http://www.zotero.org) to grab citations.
